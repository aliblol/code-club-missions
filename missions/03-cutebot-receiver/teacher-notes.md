# Teacher Notes — Mission 03: CuteBot Wireless Receiver

## Learning objectives

- Listen for radio messages in MakeCode
- Control a Cutebot using received commands
- Understand the importance of matching radio groups

## Setup

1. Open `missions/03-cutebot-receiver/` in MakeCode.
2. Ensure the Cutebot extension is available and the code is flashed.
3. Test the receiver with Mission 02 controller.

## Tips

- Remind learners that the receiver only acts on known command strings.
- Have students experiment with buttons A and B to change the radio channel.
