# Mission 03: CuteBot Wireless Receiver

This mission teaches students how to make a robot listen for radio commands and move accordingly.

## What you'll do

- Build a receiver program for the Cutebot
- Read radio commands and map them to movements
- Switch radio groups with buttons

## Files in this mission

- `pxt.json` — MakeCode package metadata
- `main.ts` — receiver code that controls the Cutebot
- `main.blocks` — blocks version for MakeCode
- `README.md` — mission instructions
- `test.ts` — optional test file

## Open in MakeCode

Open the folder in MakeCode, or import this repository URL in the editor.

## How it works

The receiver waits for strings like `forward`, `left`, `right`, `backward`, and `stop` and then uses the Cutebot extension to move the robot.

## Use with the controller

- Start Mission 02 on the controller Micro:bit.
- Flash this receiver mission to the robot.
- Ensure both Micro:bits are set to the same radio group.

## Tip

Use the A and B buttons to change the radio group and test communication reliability.
