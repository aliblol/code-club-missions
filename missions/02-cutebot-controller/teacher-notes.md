# Teacher Notes — Mission 02: CuteBot Wireless Controller

## Learning objectives

- Send radio messages between Micro:bits
- Use input events to control behavior
- Test command-based communication

## Setup

1. Open `missions/02-cutebot-controller/` in MakeCode.
2. Connect the sender Micro:bit and flash the code.
3. Have learners pair it with the receiver mission.

## Tips

- Show learners how radio groups work and why devices must match.
- Encourage them to try button combos and motion-based commands.
