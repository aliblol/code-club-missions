# Mission 02: CuteBot Wireless Controller

This mission teaches students how to send radio commands from one Micro:bit to another.

## What you'll do

- Build a sender program for the Cutebot controller
- Use buttons and motion to send direction commands
- Test sending commands to the Cutebot receiver

## Files in this mission

- `pxt.json` — MakeCode package metadata
- `main.ts` — controller code that sends radio strings
- `main.blocks` — blocks version for MakeCode
- `README.md` — mission instructions
- `test.ts` — optional test file

## Open in MakeCode

Open the folder in MakeCode, or import this repository URL in the editor.

## How it works

The sender program uses radio commands such as `forward`, `left`, `right`, `backward`, and `stop`.
The receiver mission can use those values to make the Cutebot move.

## Use with the receiver

- Start this mission on the controller Micro:bit.
- Open Mission 03 on the receiver Micro:bit.
- Make sure both devices are on the same radio group.

## Tip

Use the controller to send one command at a time, then watch how the receiver responds.
