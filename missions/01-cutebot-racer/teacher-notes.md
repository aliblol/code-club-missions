# Teacher Notes — Mission 01: CuteBot Racer

## Learning objectives

- Use event handlers and loops
- Control motors and lights
- Debug a simple robot challenge

## Setup

1. Connect the CuteBot to the computer.
2. Open the mission folder in MakeCode or use `starter.hex`.
3. Ask learners to test the baseline behavior before changing the code.

## Tips

- Encourage students to make one change at a time.
- Use the worksheet to guide reflection and testing.
