# Mission 01: CuteBot Racer

Welcome to the CuteBot Racer mission. This mission introduces programming with the CuteBot robot using MakeCode.

## What you'll do

- Explore the starter car controller
- Add speed or steering improvements
- Build a race strategy

## Files in this mission

- `starter.hex` — starter project for kids to flash to their device
- `solution.hex` — worked example
- `main.ts` — TypeScript source for the starter project
- `main.blocks` — blocks representation for MakeCode
- `worksheet.md` — mission worksheet
- `extension.md` — extra challenges

## Open in MakeCode

To import this mission into MakeCode, open the project folder and load `pxt.json` with the MakeCode editor.
